Features

Discover what makes the MIDI Generator Piano Roll special

Modern Piano Roll

Our piano roll provides a clean, modern interface for visualizing and editing MIDI notes.

  • Intuitive grid lines and time ruler
  • MIDI note visualization with velocity coloring
  • Seamless zoom functionality for detailed editing
  • Clear keyboard layout with octave indicators

Plugin Manager

Our flexible plugin system allows you to run various generation algorithms with just a few clicks.

  • Motif generation for melodic patterns
  • Markov chain algorithms for probabilistic composition
  • Custom generation logic via Python plugins
  • Smooth dockable panel with fluid animations

Dynamic Parameter Dialogs

Each plugin has its own customizable settings that allow for fine-tuned control over the generation process.

  • Plugin-specific parameters with intuitive controls
  • Real-time parameter validation
  • Save and load parameter presets
  • Tooltips and descriptive labels for easy understanding

Export to MIDI

Save your generated compositions as standard MIDI files for use in any digital audio workstation.

  • Export with velocity/pitch embedded
  • Standard .mid file format compatible with all DAWs
  • Preserve all MIDI data even if not shown in UI
  • Quick export with customizable file naming

Playback Controls

Listen to your compositions in real-time with our integrated FluidSynth audio engine.

  • Beat-synced transport controls
  • Adjustable BPM and time signature
  • Real-time instrument selection
  • Master volume control for audio level adjustment

Zoom Functionality

Easily view and edit longer note sequences with our intuitive zoom controls.

  • Horizontal and vertical zoom options
  • Zoom to fit all content
  • Focus on specific time ranges
  • Keyboard shortcuts for quick navigation
